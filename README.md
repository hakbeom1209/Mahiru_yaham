# 마히루존나야해 언어

# 0. 개요
본 문서는 마히루존나야해 언어의 기본 사양을 정의한다.  

사용 가능한 토큰은 다음과 같다.

`마`, `히`, `루`, `존`, `나`, `야`, `해`, `앙`,` 걱`

---

## 1. 기본 개념

### 1.1 숫자

숫자는 `히`의 반복으로 표현된다.

- 히 → 1  
- 히히 → 2  
- 히히히 → 3  
- 히를 n번 반복한 것은 정수 n을 의미한다.

모든 수는 정수로 처리된다.

---

### 1.2 변수

변수는 `마`의 반복으로 표현된다.

- 마 → 1번 변수  
- 마마 → 2번 변수  
- 마마마 → 3번 변수  
- 마를 n번 반복한 것은 n번 변수를 의미한다.

모든 변수는 기본값 0으로 시작한다.

---

## 2. 대입

### 2.1 기본 대입

```
마^n  히^k
```

→ n번 변수에 k를 저장한다.

예:

```
마마마 히히히
```

→ 3번 변수에 3을 저장

---

### 2.2 연산 대입

```
마^n  연산자  A  B
```

→ n번 변수에 (A 연산 B)의 결과를 저장한다.  
A와 B는 변수 또는 숫자일 수 있다.

---

## 3. 연산자

### 3.1 덧셈

```
앙 A B
```

→ A + B

---

### 3.2 곱셈

```
걱 A B
```

→ A × B

---

### 3.3 뺄셈

```
앙앙 A B
```

→ A − B

---

### 3.4 나눗셈

```
걱걱 A B
```

→ A ÷ B의 몫만 저장한다 (정수 나눗셈)

0으로 나눌 경우 결과는 0으로 처리한다.

---

## 4. 조건문

### 4.1 루 (if)

```
루 A B
```

→ A > B 일 경우 다음 한 문장을 실행한다.

---

### 4.2 존나 (블록 실행)

```
존나 <문장>
```

존나는 뒤에 오는 한 문장을 하나의 실행 단위로 묶는다.

루와 함께 사용할 경우:

```
루 A B
존나 <문장>
```

→ A > B 일 때만 <문장> 실행

---

## 5. 출력

### 5.1 야해 (ASCII 출력)

```
야해 X
```

X의 값을 ASCII 코드로 출력한다.

예:

```
마마 히히히히히
마마마 걱 마마 히히히히히히히히히히히
야해 마마마
```

5 × 13 = 65  
ASCII 65 → 'A' 출력

---

## 6. 실행 구조

- 프로그램은 위에서 아래로 순차 실행된다.
- 루가 참일 경우 존나 뒤 한 문장만 실행된다.
- 블록은 중첩되지 않는다.
- 모든 연산은 즉시 평가된다.
- 공백은 토큰 구분용이다.

---

## 7. 예제

### 예제 1: 2 + 3 출력

```
마 히히
마마 히히히
마마마 앙 마 히히히
야해 마마마
```

---

### 예제 2: 조건부 증가

```
마 히
마마 히히
루 마마 마
존나 마마 앙 마마 히히히
```

2번 변수가 1번 변수보다 클 경우  
2번 변수에 3을 더한다.
