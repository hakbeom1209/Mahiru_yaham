# 마히루존나야해 언어

## 0. 개요

본 문서는 마히루존나야해 언어의 기본 사양을 정리한다.
사용 가능한 토큰은 다음과 같다.
`마`, `히`, `루`, `존`, `나`, `야`, `해`, `앙`, `걱`

## 1. 기본 개념
### 1.1 숫자

숫자는 `히`의 반복으로 표현된다.
>히 → 1
히히 → 2
히히히 → 3

히를 n번 반복한 것은 정수 n을 의미한다.
모든 수는 기본적으로 정수로 처리된다.

## 2. 대입
### 2.1 기본 대입

>`마` ^n  `히` ^k

는 n번 변수에 k를 저장한다.

예를들어
> 마마마 히히히

는 3번 변수에 3을 저장한다

### 2.2 연산 대입

>`마`^n  연산자  A  B

는 n번 변수에 (A 연산 B)의 결과를 저장한다.

A와 B는 변수 또는 숫자일 수 있다.

### 3. 연산자
## 3.1 덧셈

> 앙 A B

→ A + B

## 3.2 곱셈

> 걱 A B

→ A × B

## 3.3 뺄셈

> 앙앙 A B

→ A − B

3.4 나눗셈

> 걱걱 A B

→ A ÷ B의 몫만 저장 (정수 나눗셈)

0으로 나눌 경우 결과는 0으로 처리한다.
